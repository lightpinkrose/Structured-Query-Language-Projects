# Structured-Query-Language-Projects
SQL-based relational database projects that demonstrate joins, subqueries, aggregations, data transformation, and reporting queries.
Overview
This project demonstrates relational database querying and reporting techniques using structured SQL. The database models a university-style system including courses, sections, students, instructors, and enrollments.

The focus of this project was to apply advanced SQL concepts to retrieve, transform, and analyze structured relational data.
